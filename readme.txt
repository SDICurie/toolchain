This tarball contains command line only toolchains taken from Intel(R) System Studio 2016 for Microcontrollers:
https://software.intel.com/en-us/intel-system-studio-microcontrollers

Intel(R) System Studio 2016 for Microcontrollers is an Eclipse*-integrated software tool set for developing, debugging, and optimizing systems and firmware for the Intel(R) Quark(TM) microcontroller D2000 and Intel(R) Quark(TM) SE microcontroller.

Download the Intel(R) System Studio 2016 for Microcontrollers from the Intel(R) Software Development Products Registration Center page:
https://registrationcenter.intel.com/en/forms/?productid=2580
